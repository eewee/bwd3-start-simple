# bwd3-start-simple

## Variable

Utilisez des variables pour remplacer : 
- Adresse `$address`
- Téléphone `$phone`
- Email `$email`

## Tableau
- Créer un tableau nommé "$stories".
- Ajouter dans ce tableau, 3 stories.
- Composition d'une storie :
```
[
    "image"       => "url de votre image",
    "description" => "description de votre storie"
]
```

## Boucle
- Réaliser une boucle foreach sur le tableau $stories.
- Afficher les 3 stories dans la section nommée "OUR SUCCESS STORIES".

## Formulaire

- Récupérer les valeurs du formulaire, envoyée en POST, lors de la validation de ce dernier et les afficher juste au-dessus du formulaire.
